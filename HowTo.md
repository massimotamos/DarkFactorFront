# How to Start the Tool

This project is an Angular frontend application.

## Prerequisites

- Node.js 20 or newer
- npm

## Install dependencies

Run this once in the project root:

```powershell
npm install
```

## Start the development server

Run:

```powershell
npm start
```

This uses Angular's dev server (`ng serve`) and starts the application in development mode.

After startup, open:

```text
http://localhost:4200
```

## Other useful commands

Build the project:

```powershell
npm run build
```

Run tests:

```powershell
npm test
```

Watch mode build:

```powershell
npm run watch
```

## Notes

- Run all commands from the project root: `DarkFactorFront`
- If `npm start` fails because dependencies are missing, run `npm install` first
